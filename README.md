# Intro Component with Sign Up Form

This project is an implementation of JavaScript form validation.

It ensures the user fills in all required fields. If any field is missing or invalid, an error is shown instead of submitting the form.

## What It Does

- Checks each form input on submit
- Prevents submission when any required field is empty
- Shows an error message for invalid fields
- Submits only when all fields pass validation

## Tech Used

- HTML
- CSS
- JavaScript
